# 代码结构优化说明

## 优化内容

### 1. 模块化架构 - Flask Blueprints
将原来 638 行的 `app.py` 拆分为 9 个独立的 Blueprint 模块：

- **api/power.py** - 强电控制 + 主页面 (/, /config)
- **api/light.py** - 场馆灯光控制
- **api/door.py** - 门禁与监控 (含 OpenCV 视觉检测)
- **api/server.py** - 服务器看板 (含 SQLite 数据库)
- **api/projector.py** - 投影机集群控制
- **api/screen.py** - 幕布控制
- **api/universal.py** - 泛型设备控制
- **api/env.py** - 环境监测
- **api/automation.py** - 自动化运行

### 2. 清晰的目录结构

```

### 3. 优势

#### 易于扩展
- 新增功能只需创建新的 Blueprint 文件
- 在 `app.py` 中注册即可，无需修改其他代码

#### 代码清晰
- 每个模块职责单一，代码量小 (50-150行)
- 相关功能集中在一起，易于维护

#### 团队协作
- 不同开发者可以并行开发不同模块
- 减少代码冲突

```

2. 在 `app.py` 中注册:
```python

app.register_blueprint(hvac_bp)
```
