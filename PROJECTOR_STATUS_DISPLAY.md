# 投影机状态显示功能说明

## 功能概述

已成功为投影机控制系统添加实时状态显示功能，支持以下状态信息的查询和展示:

- ✅ **电源状态**: 开机/关机/未知
- 🌡️ **温度**: 实时温度监控 (高温告警 >60℃)
- ⏱️ **灯泡时长**: 累计使用时长 (小时)
- 📺 **信号源**: 当前输入信号源 (HDMI/VGA/Video/USB/LAN 等)

## 技术实现

### 1. 后端 API

**接口**: `/api/projector/status` (GET)

**返回格式**:
```json
{
  "projector_1": {
    "online": true,
    "power": "on",
    "temp": 45,
    "lamp_hours": 1234,
    "source": "HDMI"
  },
  "projector_2": {
    "online": false,
    "power": "unknown",
    "temp": null,
    "lamp_hours": null,
    "source": null
  }
}
```

### 2. 前端显示

- **位置**: 主页 -> 投影机集群快捷控制下方
- **刷新频率**: 每 5 秒自动刷新
- **显示样式**: 卡片式布局，每台投影机独立卡片
- **状态指示**: 
  - 🟢 在线 (绿色边框)
  - ⚫ 离线 (灰色边框)

## 状态指示说明

### 电源状态
- 🔌 **开机** (绿色): 投影机处于开启状态
- ⏻ **关机** (灰色): 投影机处于关闭状态
- ❓ **未知** (黄色): 无法确定状态

### 温度状态
- 🌡️ **正常** (绿色): 温度 ≤ 60℃
- 🌡️ **高温** (红色): 温度 > 60℃ (需要关注)
- 🌡️ **--** (灰色): 不支持温度查询或查询失败

### 在线状态
- 🟢 **在线** (绿色): 通讯正常
- ⚫ **离线** (灰色): 通讯失败

## 注意事项

1. **刷新频率**: 默认每 5 秒刷新一次，过于频繁可能影响性能
2. **超时处理**: 查询失败时显示灰色"--",不会影响其他功能
3. **信号源映射**: 可根据实际投影机型号在 `projector_core.py` 中调整映射表
4. **多投影机**: 支持同时显示多台投影机状态，每台独立卡片

## 故障排查

### 状态显示"离线"
- 检查投影机网络连接
- 检查串口服务器状态
- 验证 IP 地址和端口配置
- 查看系统日志

### 部分状态显示"--"
- 该型号投影机可能不支持此功能
- 查询命令可能不匹配
- 检查投影机是否处于待机状态

### 状态更新缓慢
- 网络延迟可能导致
- 投影机响应慢
- 增加刷新间隔时间
