# Windows 直接运行说明

## 环境准备

项目已配置好 Python 虚拟环境 (`.venv`)，所有依赖已安装。

## 启动步骤

### 方式一：使用启动脚本（推荐）

双击运行 `start.bat` 即可启动服务。

### 方式二：手动启动

```bash
# 激活虚拟环境
.venv\Scripts\activate

# 运行应用
python app.py
```

## 访问地址

启动后访问: http://localhost:5000

## 注意事项

1. 确保端口 5000 未被占用
2. 所有配置在 `config.json` 中
3. 日志文件：`operation_logs.json` 和 `energy_log.json`
4. 时区使用系统时区（无需额外配置）

## 停止服务

在命令行窗口按 `Ctrl+C` 停止服务。
