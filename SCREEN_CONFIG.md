# 🎬 幕布配置信息

## ✅ 已更新的实际配置

### 网络配置
- **IP 地址**: `192.168.50.76`
- **端口**: `1234`
- **协议**: TCP

### 控制指令

| 功能 | 名称 | 16 进制指令 | 动作类型 |
|------|------|------------|---------|
| ⬆️ 上升 | 幕布上升 | `FF AA EE EE DD` | up |
| ⏹️ 停止 | 幕布停止 | `FF AA EE EE CC` | stop |
| ⬇️ 下降 | 幕布下降 | `FF AA EE EE EE` | down |

---

## 📋 使用步骤

### 1. 添加幕布设备

1. 进入**系统总配置中心**
2. 点击**🎬 幕布升降**选项卡
3. 点击 **"+ 注册幕布设备"**

### 2. 填写配置

```
显示名称：幕布升降控制
品牌：幕布升降控制
系列：幕布升降
控制类型：幕布控制 (TCP 网口)
IP 地址：192.168.50.76
端口：1234
```

### 3. 配置幕布参数

**需要实测**:
```
幕布总高度 (米): ___  (用卷尺测量)
全程升降时间 (秒): ___  (用秒表计时)
```

### 4. 加载命令

点击 **"📥 一键加载标准命令集"**

系统会自动加载 3 个命令:
- ⬆️ 幕布上升 (`FF AA EE EE DD`)
- ⏹️ 幕布停止 (`FF AA EE EE CC`)
- ⬇️ 幕布下降 (`FF AA EE EE EE`)

### 5. 位置标定

**首次使用必须标定**:

1. **0% 标定**:
   - 按"上升"直到幕布完全收起
   - 选择"完全收起 (0%)"
   
2. **100% 标定**:
   - 按"下降"直到幕布完全下降
   - 选择"完全下降 (100%)"

### 6. 保存配置

点击 **"💾 批量保存并生效"**

## ⚠️ 注意事项

### 1. 必须实测的参数
- **幕布总高度**: 从完全收起到完全下降的垂直距离
- **全程升降时间**: 从按下降键到完全停止的时间

### 2. 标定重要性
- ⚠️ 未标定会导致位置显示不准确
- 建议每周标定一次
- 调整后需重新标定

### 3. 网络配置
- 确保上位机与幕布控制器在同一网段
- 确保端口 1234 未被占用
- 确保网络连接正常

---

## 🐛 故障排查

### 控制无反应
1. 检查 IP 地址是否正确 (192.168.50.76)
2. 检查端口是否正确 (1234)
3. 检查网络连接
4. 查看系统操作日志

### 位置显示不准
1. 重新标定 0% 和 100%
2. 实测并更新全程时间
3. 多次测试验证
