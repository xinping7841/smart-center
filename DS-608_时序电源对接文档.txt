DS-608 时序电源对接文档

一、设备基础信息
1. 品牌：DGH
2. 型号：DS-608
3. 编号：DS-608
4. 类型：时序器
5. 材质：铝合金
6. 颜色：图片色
7. 通道数量：8路

二、接入方式
1. TCP/IP
IP：192.168.50.53
端口：8080

2. RS232
波特率：19200
数据位：8
停止位：1
校验位：NONE

三、协议概述
本设备常规控制采用 6 字节短帧格式：
55 + 地址 + 命令 + 数据1 + 数据2 + 校验

说明：
1. 帧头固定为 0x55
2. 地址为设备 ID，常用 0x01、0x02、0x03
3. 校验为异或校验
4. 校验算法：
   CS = 地址 XOR 命令 XOR 数据1 XOR 数据2
5. 帧头 0x55 不参与校验

四、常用命令格式
1. 开启某一路
格式：
55 ID A0 00 CH CS

2. 关闭某一路
格式：
55 ID A1 00 CH CS

3. 顺序开启全部
格式：
55 ID A4 00 00 CS

4. 逆序关闭全部
格式：
55 ID A5 00 00 CS

5. 所有通道同时开启
单机：
55 ID A2 00 00 CS
广播：
55 00 AC 00 00 AC

6. 所有通道同时关闭
单机：
55 ID A3 00 00 CS
广播：
55 00 AD 00 00 AD

7. 状态查询
55 ID A8 00 00 CS

8. 机器查询
55 ID A9 00 00 CS

9. 读取机器地址
55 00 AB 00 00 AB

六、状态返回说明
状态查询命令：
55 ID A8 00 00 CS

状态返回格式：
57 ID B0 D1 D2 D3 CS

字段说明：
1. D1 为系统状态字节
   bit0：LCD 状态，0=开，1=关
   bit1：锁定状态，0=未锁定，1=锁定
   bit2：运行过程状态位

2. D2 为 8 路输出状态位图
   b7 到 b0 对应 CH1 到 CH8
   1=关闭
   0=开启

3. D3 为扩展状态位
   目前可知与屏蔽/非屏蔽状态有关

七、模式相关
设备支持三种模式：
1. 时序模式
2. 互锁模式
3. 继电器模式

模式说明：
1. 时序模式下支持单路控制、顺序控制、级联、锁定、定时等指令
2. 互锁模式下共4组互锁：
   1-2、3-4、5-6、7-8
3. 继电器模式下仅支持设备本体操作

八、级联与锁定
1. 级联保存：55 01 A6 02 00 A5
2. 解除级联保存：55 01 A6 05 00 A2
3. 级联后顺序开启/关闭：55 01 A6 07 00 A0
4. 级联定时开：55 01 A6 08 00 AF
5. 级联定时关：55 01 A6 09 00 AE
6. 锁定保存：55 01 AE AA 00 05
7. 不锁定保存：55 01 AE 55 00 FA

九、建议对接方式
1. 主轮询命令优先使用状态查询 A8
2. 初次识别可使用读地址 AB 和机器查询 A9
3. TCP 与 RS232 建议共用同一套协议编码器
4. 现阶段系统内已补充 DS-608 模拟控制界面，便于先联调页面与交互
