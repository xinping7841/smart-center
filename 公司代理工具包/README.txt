Company Proxy Tool

1. Extract the archive.
2. Run 公司代理-一键部署.cmd first.
3. After deployment, desktop shortcuts will be created automatically:
   - 公司代理-开启
   - 公司代理-关闭
   - 公司代理-诊断
4. Use the desktop shortcuts afterward.

Proxy Server: 192.168.50.254:7895
Install Path: %LOCALAPPDATA%\CompanyProxyTool

If enable fails, run the diagnose shortcut and send the screenshot back.
If you need to remove the tool, run 公司代理-一键清除.cmd.
