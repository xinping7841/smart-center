# 投影机快速配置指南

## 🎯 立即开始

### 方式 1: 视美乐投影机 (通过串口服务器)

**适用场景**: 投影机通过 RS232 连接到串口服务器

**配置步骤**:

1. 访问配置页面: http://localhost:6899/config

2. 添加投影机配置:
   ```
   品牌系列：视美乐 EK 系列
   设备 ID: 30  ← 重要！根据实际投影机 ID 设置
   通讯协议：视美乐 EK 系列 (TCP 网口 / 串口服务器透传)
   IP 地址：192.168.50.72
   端口：502
   ```

3. 点击"📥 一键加载标准命令集"

4. 点击"💾 批量保存并生效"

5. 回到主页测试开机/关机

**指令格式说明**:
- 配置 ID=30 时，开机指令：`#PWR30,01` (16 进制：`23 50 57 52 33 30 2C 30 31`)
- 配置 ID=1 时，开机指令：`#PWR1,01` (16 进制：`23 50 57 52 31 2C 30 31`)
- 系统会自动替换 ID，无需手动修改指令

---

### 方式 2: 光峰 HD500 (网络直连)

**适用场景**: 投影机直接连接网络

**配置步骤**:

1. 访问配置页面: http://localhost:6899/config

2. 添加投影机配置:
   ```
   品牌系列：光峰 HD500
   通讯协议：光峰 HD500 (TCP 网口)
   IP 地址：192.168.50.110  ← 投影机 IP
   端口：10000  ← 光峰默认端口
   ```

3. 点击"📥 一键加载标准命令集"

4. 点击"💾 批量保存并生效"

5. 回到主页测试开机/关机

**协议说明**:
- 使用 15 字节二进制协议
- 包含校验和计算
- 开机指令：`EB 90 00 08 01 00 01 00 00 01 00 00 00 01 87`

## ❓ 常见问题

### Q1: 视美乐投影机 ID 在哪里设置？

A: 在投影机菜单中:
1. 按投影机遥控器"MENU"键
2. 进入"设置" → "控制设置" → "设备 ID"
3. 设置 ID 值 (默认通常是 30)
4. 在系统配置页面设置相同的 ID

### Q2: 如何确认指令是否发送成功？

A: 查看系统日志窗口:
- 如果显示"✅ 数据已发送" → 网络通讯正常
- 如果显示"⚠️ 读取超时" → 正常，串口服务器不返回确认
- 观察投影机是否有实际反应

### Q3: 光峰投影机连接被拒绝？

A: 检查:
- IP 地址是否为 192.168.50.110
- 端口是否为 10000
- 投影机是否已开机
- 网络是否通畅 (ping 192.168.50.110)

### Q4: 配置后不生效？

A: 确保:
1. 点击了"💾 批量保存并生效"按钮
2. 刷新了主页 (F5)
3. 检查配置页面的 IP/端口/ID 是否正确
