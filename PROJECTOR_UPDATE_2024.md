# 投影机控制协议更新说明

## ✅ 已完成的更新

### 1. 视美乐 EK 系列 - 添加设备 ID 配置

**问题**: 视美乐投影机需要通过设备 ID 进行控制，原指令格式不正确

**解决方案**:
- ✅ 修改指令格式为：`#PWR{ID},{命令}`
- ✅ 添加设备 ID 配置选项 (默认值：30)
- ✅ 实现动态 ID 替换功能

**指令格式对比**:

| 功能 | 原指令 (错误) | 新指令 (正确) | 16 进制 (ID=30) |
|------|--------------|--------------|----------------|
| 开机 | `#1POWR 1` | `#PWR30,01` | `23 50 57 52 33 30 2C 30 31` |
| 关机 | `#1POWR 0` | `#PWR30,00` | `23 50 57 52 33 30 2C 30 30` |
| HDMI1 | `#1SOUR 17` | `#SOUR30,17` | `23 53 4F 55 52 33 30 2C 31 37` |

**配置方法**:

1. 访问配置页面：`http://localhost:6899/config`
2. 进入"🎥 投影机设备"
3. 选择品牌：**视美乐 EK 系列**
4. 设置**设备 ID (十进制)**: 输入投影机的实际 ID (默认 30)
5. 点击"💾 批量保存并生效"

**ID 替换示例**:

```python
# 配置设备 ID = 1
# 开机指令自动变为：#PWR1,01
# 16 进制：23 50 57 52 31 2C 30 31

# 配置设备 ID = 99
# 开机指令自动变为：#PWR99,01
# 16 进制：23 50 57 52 39 39 2C 30 31
```

## 📝 注意事项

### 视美乐投影机

1. **设备 ID 必须匹配**: 确保配置的 ID 与投影机实际 ID 一致
2. **指令格式**: 使用 `#PWR{ID},{命令}` 格式，不是 `#1POWR 1`
3. **ID 范围**: 通常是 0-99，具体参考投影机手册
4. **默认 ID**: 大多数视美乐投影机默认 ID 为 30

### 光峰 HD500

1. **网络连接**: 确保上位机能访问 192.168.50.110
2. **端口开放**: 确认投影机 10000 端口开放
3. **协议格式**: 使用 15 字节二进制协议 (含校验和)
4. **无确认机制**: 发送后不等待响应是正常的

## 🎯 故障排查

### 视美乐投影机无法控制

**问题 1: 指令格式错误**
- 检查日志中的 16 进制数据
- 应该包含设备 ID (如 33 30 表示"30")
- 如果不包含，检查配置页面的设备 ID 设置

**问题 2: ID 不匹配**
- 确认投影机实际 ID
- 在配置页面修改设备 ID
- 保存后重试

**问题 3: 网络不通**
- 测试 ping 串口服务器 IP
- 测试 telnet 端口是否开放
- 检查串口服务器配置

### 光峰投影机无法控制

**问题 1: 连接被拒绝**
- 检查 IP 地址是否正确 (应该是 192.168.50.110)
- 检查端口是否为 10000
- 确认投影机已开机

**问题 2: 无响应**
- 光峰协议可能不返回确认
- 只要投影机有反应就说明成功
- 查看系统日志确认数据已发送
