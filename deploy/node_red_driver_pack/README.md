# Node-RED Driver Pack

This folder is generated by script. It helps import current driver data into Node-RED.

## Files
- `flows_driver_hub.json`: import this flow in Node-RED
- `drivers_manifest.json`: current driver inventory
- `snapshot_example.json`: current real-time snapshot sample

## Import
1. Open Node-RED editor.
2. Menu -> Import -> choose `flows_driver_hub.json`.
3. Deploy.
4. If your backend URL is different, edit node `get driver snapshot` to:
   `http://127.0.0.1:6899/api/driver_hub/snapshot`

## Notes
- Data source is `/api/driver_hub/snapshot` from this project.
- UI page for same data: `/driver_hub`.
