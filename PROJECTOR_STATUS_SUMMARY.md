## 🔧 技术细节

### 信号源映射
```python
source_map = {
    "0": "无信号",
    "1": "HDMI",
    "2": "VGA",
    "3": "Video",
    "4": "USB",
    "5": "LAN"
}
```

### 状态查询流程
1. 前端定时调用 `/api/projector/status`
2. 后端遍历所有投影机配置
3. 对每台投影机执行:
   - 查询电源状态 (`get_power_status`)
   - 查询温度 (`get_temp`)
   - 查询灯泡时长 (`get_lamp_hours`)
   - 查询信号源 (`get_source`)
4. 返回 JSON 格式状态数据
5. 前端渲染状态卡片

### 状态指示颜色
- **绿色** (`var(--success)`): 正常/开机/在线
- **灰色** (`var(--text-sub)`): 关机/离线/不支持
- **红色** (`var(--danger)`): 高温告警
- **黄色** (`var(--warning)`): 未知状态

## 📝 使用说明

### 查看投影机状态
1. 打开系统主页
2. 滚动到投影机控制区域
3. 查看"投影机实时状态"面板
4. 状态自动刷新，无需手动操作

### 解读状态信息
- **在线状态**: 绿色边框表示在线，灰色表示离线
- **电源**: 开机 (绿色)/关机 (灰色)/未知 (黄色)
- **温度**: 绿色表示正常，红色表示高温 (>60℃)
- **灯泡时长**: 蓝色数字显示
- **信号源**: 显示当前输入源名称
