# 投影机配置更新总结

## ✅ 更新内容

### 1. 品牌和系列分离

**更新前:**
- 品牌系列混在一起 (如"光峰 HD500")
- 无法清晰区分品牌和系列

**更新后:**
- 品牌单独选择：光峰、视美乐
- 系列单独选择：DH 系列、EK 系列
- 层级清晰：品牌 → 系列 → 连接类型

### 2. 删除 HD500，只保留 DH 系列

- ✅ 删除所有 HD500 相关配置
- ✅ 只保留 DH 系列 (光峰)
- ✅ 只保留 EK 系列 (视美乐)

### 3. 连接类型分离

**网口和串口不再混显:**
- 选择 TCP/UDP 时只显示 IP 和端口配置
- 选择 COM 时只显示串口和波特率配置

### 4. 默认配置更新

**新增投影机默认值:**
- 品牌：appotronics (光峰)
- 系列：dh (DH 系列)
- 控制类型：appotronics_dh_tcp (TCP 网口)
- IP: 192.168.50.110
- 端口：9761

## 📊 配置结构

```
品牌 (Brand)
├── 光峰 (appotronics)
│   └── DH 系列 (dh)
│       ├── TCP 网口 (appotronics_dh_tcp) - 端口 9761
│       ├── UDP 网口 (appotronics_dh_udp) - 端口 9761
│       └── 串口 COM (appotronics_dh_com) - 波特率 9600
└── 视美乐 (smile)
    └── EK 系列 (ek)
        ├── TCP 网口 (smile_ek_tcp)
        ├── UDP 网口 (smile_ek_udp)
        └── 串口 COM (smile_ek_com)
```

## 🔧 修改的文件

1. **templates/config.html**
   - 品牌下拉框：只显示光峰、视美乐
   - 系列下拉框：根据品牌动态显示 DH/EK 系列
   - 控制类型：根据品牌和系列动态显示
   - 端口配置：根据连接类型动态显示 IP/COM
   - 新增 `loadSeriesCommands` 函数
   - 更新 `addProj` 默认值

2. **app.py**
   - 新增 `api_projector_brand_commands` 兼容接口

3. **projector_core.py**
   - 保持三层结构函数不变

## 🎯 使用方式

### 配置新投影机

1. 点击"+ 注册投影机"
2. 选择品牌：光峰
3. 选择系列：DH 系列
4. 选择连接类型：
   - TCP 网口 → 填写 IP 和端口 (9761)
   - 串口 COM → 填写 COM 端口和波特率 (9600)
5. 点击"一键加载标准命令集"
6. 选择命令格式 (16 进制/字符串)
7. 保存配置

### 加载 DH 系列命令

```javascript
// 前端调用
loadSeriesCommands(投影机索引，'appotronics', 'dh')

// API 请求
GET /api/projector/series_commands?brand_id=appotronics&series_id=dh
```
