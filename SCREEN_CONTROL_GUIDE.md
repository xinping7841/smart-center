# 🎬 幕布升降控制系统 - 使用指南

## ✅ 功能概述

已实现**简单模式**的幕布升降控制，包含以下功能:

### 核心功能
- ✅ **三键控制**: 升、停、降
- ✅ **时间计算**: 基于升降时间自动计算位置
- ✅ **实时显示**: 百分比 + 高度 (米) + 状态
- ✅ **手动标定**: 支持 0%、50%、100% 位置标定
- ✅ **状态持久化**: 断电后位置不丢失
- ✅ **进度条可视化**: 直观显示当前位置

### 技术特性
- 🔄 **自动刷新**: 每 3 秒自动更新状态
- 📊 **进度显示**: 渐变色进度条
- ⏱️ **时间计算**: 精确到 0.1 秒
- 💾 **配置保存**: 自动保存到 screens_config.json

## 📋 配置步骤

### 1️⃣ 添加幕布设备

1. 进入**系统总配置中心**
2. 点击**🎬 幕布升降**选项卡
3. 点击 **"+ 注册幕布设备"**
4. 填写配置:
   ```
   内部 ID: screen_001 (自动生成)
   显示名称：主会场幕布
   品牌：幕布升降控制 (固定)
   系列：幕布升降 (固定)
   ```

### 2️⃣ 配置通讯方式

选择通讯协议:
- **TCP 网口**: 适用于网络控制器
  - IP 地址：192.168.x.x
  - 端口：502 (默认)
- **串口 COM**: 适用于串口控制器
  - COM 端口：COM1
  - 波特率：9600

### 3️⃣ 配置幕布参数

**重要参数** (需要实测):
```
幕布总高度 (米): 3.0     # 从完全收起到完全下降的垂直高度
全程升降时间 (秒): 30    # 从完全收起到完全下降所需时间
```

**计算方法**:
1. 用卷尺测量幕布完全收起和完全下降的垂直距离
2. 用秒表计时，从按下降键到幕布完全停止的时间

### 4️⃣ 位置标定

**首次使用必须标定**:

1. **0% 标定** (完全收起):
   - 按"上升"键直到幕布完全收起
   - 等待 3 秒确保完全停止
   - 在配置页面选择"完全收起 (0%)"
   - 系统自动保存当前位置

2. **100% 标定** (完全下降):
   - 按"下降"键直到幕布完全下降
   - 等待 3 秒确保完全停止
   - 在配置页面选择"完全下降 (100%)"
   - 系统自动保存当前位置

3. **验证标定**:
   - 查看"当前高度"显示是否正确
   - 0% 时应为 0.00 米
   - 100% 时应为总高度 (如 3.0 米)

### 5️⃣ 加载标准命令

1. 点击 **"📥 一键加载标准命令集"**
2. 系统自动加载 3 个标准命令:
   - ⬆️ 幕布上升
   - ⏹️ 幕布停止
   - ⬇️ 幕布下降

### 6️⃣ 配置命令显示

每个命令可配置:
- **按键名称**: 如"幕布上升"
- **16 进制指令**: 根据实际控制器协议修改
- **动作类型**: up / stop / down
- **指令格式**: hex(16 进制) / str(字符串)
- **大屏主页显示**: 勾选后在主页显示快捷按钮
- **排序**: 数字越小越靠前

**默认指令** (需根据实际情况修改):
```
幕布上升：AA 01 01 01
幕布停止：AA 01 01 03
幕布下降：AA 01 01 02
```

## 🔧 高级配置

### 修改控制指令

如果您的幕布控制器使用不同的指令:

1. 在配置页面找到对应命令
2. 修改"16 进制指令"
3. 确保"动作类型"正确
4. 保存配置

**常见指令格式**:
```
格式 1 (16 进制):
  上升：AA 01 01 01
  停止：AA 01 01 03
  下降：AA 01 01 02

格式 2 (字符串):
  上升：UP
  停止：STOP
  下降：DOWN

格式 3 (Modbus):
  上升：01 05 00 00 FF 00
  停止：01 05 00 01 FF 00
  下降：01 05 00 02 FF 00
```

可以添加更多功能:
- 预设位置 (如 30%、50%、70%)
- 微调命令 (上升 5 秒、下降 5 秒)
- 特殊功能 (灯光、锁定等)

**添加步骤**:
1. 点击"+ 增加幕布控制按键"
2. 填写配置:
   ```
   按键名称：预设位置 50%
   16 进制指令：AA 01 02 32 (示例)
   动作类型：stop
   指令格式：hex
   大屏主页显示：是
   排序：10
   ```

## ⚠️ 注意事项

### 1. 标定重要性
- ⚠️ **必须标定**才能准确显示位置
- 建议每周标定一次
- 更换幕布或调整后需重新标定

### 2. 时间计算误差
- 电机电压变化可能影响速度
- 温度变化可能影响机械性能
- 建议定期验证实际位置

### 3. 网络延迟
- TCP 控制时注意网络稳定性
- 如果延迟高，建议增加超时时间
- 串口控制更稳定但距离受限

### 4. 状态同步
- 系统通过时间计算位置，非真实反馈
- 长时间使用后建议重新标定
- 手动干预后需更新状态

## 🐛 故障排查

### 问题 1: 位置显示不准确

**原因**:
- 未标定或标定不准确
- 幕布实际速度与配置不符

**解决方法**:
1. 重新标定 0% 和 100%
2. 实测全程时间并更新配置
3. 多次测试验证

### 问题 2: 控制无反应

**原因**:
- 网络连接中断
- IP/端口配置错误
- 指令格式不正确

**解决方法**:
1. 检查网络连接
2. 验证 IP 和端口配置
3. 查看系统操作日志
4. 用网络助手测试指令

### 问题 3: 状态不刷新

**原因**:
- 前端 JavaScript 错误
- API 接口异常

**解决方法**:
1. 按 F12 查看浏览器控制台
2. 检查后端服务是否运行
3. 刷新页面重试
