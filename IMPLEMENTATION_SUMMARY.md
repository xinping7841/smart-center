## 🚀 使用方法

### 快速开始:

1. **启动服务器**
   ```bash
   python app.py
   ```

2. **访问配置页面**
   ```
   http://localhost:6899/config
   ```

3. **添加投影机**
   - 点击"🎥 投影机设备"标签
   - 点击"+ 注册投影机"
   - 选择品牌 (如"视美乐 EK 系列")
   - 点击"📥 一键加载标准命令集"
   - 配置 IP/端口或 COM 端口
   - 调整需要显示的命令
   - 点击"💾 批量保存并生效"

4. **使用主页控制**
   ```
   http://localhost:6899/
   ```
   - 查看 Dashboard 投影机快捷按钮
   - 点击按钮发送指令
   - 查看系统日志窗口

## ⚠️ 注意事项

1. **首次使用**:
   - 确保选择正确的品牌
   - 检查通讯参数 (IP/端口/波特率)

2. **网络控制**:
   - 确保投影机和服务器在同一网络
   - 检查防火墙设置

3. **串口控制**:
   - 确保 COM 端口未被占用
   - 波特率需匹配 (视美乐 EK 默认 19200)

4. **命令格式**:
   - Hex 格式：空格分隔的字节对 (如 `23 31 50 4F`)
   - 字符串格式：直接输入文本 (如 `#1POWR 1`)
