## 🔧 API 接口

### 控制接口
```
POST /api/screen/control
Body: {
  "screen_id": "screen_001",
  "command": {
    "payload": "AA 01 01 01",
    "format": "hex",
    "action": "up"
  }
}
```

### 状态接口
```
GET /api/screen/status?screen_id=screen_001
Response: {
  "position": 75.0,
  "height": 2.25,
  "action": "stop",
  "is_moving": false,
  "remaining_time": 0,
  "total_height": 3.0
}
```

### 标定接口
```
POST /api/screen/calibrate
Body: {
  "screen_id": "screen_001",
  "position": 0  // 0-100
}
```
